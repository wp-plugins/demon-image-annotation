Put the plugin folder into [wordpress_dir]/wp-content/plugins/2. Go into the WordPress admin interface and activate the plugin3. Complete usage instructions are available here. (http://www.superwhite.cc/demon/image-annotation-plugin)Usage:- To embed annotations and comments on images, you must insert id attribute start with 'img-' and follow by uniqe id.- All the images from my blog is from flickr, I always grab the last parameters from flickr link as id which is uniqe. http://www.flickr.com/photos/30601564@N04/4825822806/- Make sure all the images have unqie and different id else you will get the same comments.Example:<img id="img-4774005463" src="http://farm5.static.flickr.com/4121/4774005463_3837b6de44_o.jpg" width="900" height="599" alt="Image Annotation Plugin" />Optional:To add thumbnails to your comments list, just add the php code below in your comment custom function. (*Note this only work with custom function)<?php echo getImgID($comment->comment_ID); ?>== Frequently Asked Questions ==No questions have been asked.== Screenshots ==No screenshots are available.
