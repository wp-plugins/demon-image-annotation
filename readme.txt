Put the plugin folder into [wordpress_dir]/wp-content/plugins/2. Go into the WordPress admin interface and activate the plugin3. Complete usage instructions are available here. (http://www.superwhite.cc/demon/image-annotation-plugin)== Frequently Asked Questions ==No questions have been asked.== Screenshots ==No screenshots are available.== Changelog === 1.1   =* Fix note overlap* Image note user addable= 1.2   =* Delete comments* Comment thumbnail hover
